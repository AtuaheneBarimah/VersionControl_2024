1. Shebang: #!/bin/bash
2. echo <argu>
3. cat <argu>
4. executable file: 
	chmod u+x <argu>
5. Working Directory: pwd
6. Create a file named hello_world.sh
	touch hello_world.sh
7. Find the path to your bash shell.
	which bash
8. How to define variables

	#!/bin/bash
	# A simple variable example
	greeting=Hello
	name=Tux
	echo $greeting $name

9. Arithmetic Expressions
	#!/bin/bash
	var=$((3+9))
	echo $var

10. How to read user input
	#!/bin/bash
	echo "Enter a number"
	read a
	echo "Enter a number"
	read b
	var=$((a+b))
	echo $var
